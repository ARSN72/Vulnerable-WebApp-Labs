What is Brute Force? Brute force attacks try multiple username/password combinations until they find the correct one.

💡 How Brute Force Works:

  1. Attacker tries common usernames (admin, user, test)
  2. Uses password lists (123456, password, admin)
  3. Automates the process with tools
  4. Exploits lack of rate limiting

🛡️ How to Prevent Brute Force:

  • Implement rate limiting (max 5 attempts per IP)
  • Use CAPTCHA after failed attempts
  • Account lockout after multiple failures
  • Strong password policies
  • Two-factor authentication (2FA)
